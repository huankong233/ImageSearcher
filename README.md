# ImageSearcher

写了给我机器人用的搜图库
具体如何使用可以看`test.js`

## 🚀支持情况

所有不推荐都不是原生支持
| 搜索引擎  | url      | imagePath |
| -------- | -------  | --------- |
| ascii2d  | ✔        | ✔        |
| EHentai  | ✔(不推荐) |✔         |
| IqDB     | ✔        |✔         |
| SauceNAO | ✔        |✔         |
| TraceMoe | ✔(不推荐) |✔         |
| Yandex   | ✔        |❌        |

## 🎉使用方法

1.克隆项目

~~~sh
git clone https://github.com/huankong233/ImageSearcher.git
~~~

2..安装支持库

~~~sh
yarn install
~~~

5.运行框架

~~~sh
yarn test
~~~
