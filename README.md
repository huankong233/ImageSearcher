# ImageSearcher
写了给我机器人用的搜图库

# 🎉使用方法
1.克隆项目
~~~sh
git clone https://github.com/huankong233/ImageSearcher.git
~~~

2..安装支持库
~~~sh
yarn install
~~~

5.运行框架
~~~sh
yarn test
~~~
